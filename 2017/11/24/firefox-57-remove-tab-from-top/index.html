파이어 폭스 57 상당탭 제거하기

우선 상단 탭이 제거된 Firefox의 모습입니다.

물론 상단탭이 제거되면 불편하겠지만, 저는 Tree Style Tab이라는 Add-on을 애용하기 때문에 큰 불편함이 없습니다.

오히려 Tree Style Tab을 사용하게 되면 탭에 상단과 좌 또는 우측에 중복으로 나오는 문제가 있습니다.

주로 사용되는게 좌/우측의 탭이므로 상단탭 제거를 시도해봤습니다.

방법은 의외로 간단합니다.

  1. Firefox의 profile 폴더를 확인합니다.
    Profile folder - Firefox
    윈도우 기준으로 봤을 때 %APP_DATA%가 가리키는 곳을 기준으로 찾아가시면 됩니다.

  2. profile 폴더 내에서 chrome 폴더를 확인합니다. 만약에 해당 폴더가 없을 경우, 새로 생성하시면 됩니다.

  3. chrome폴더 내에 userChrome.css 파일을 생성 후, 아래의 내용을 추가합니다.

  4. 파이어폭스를 재시작 합니다.
